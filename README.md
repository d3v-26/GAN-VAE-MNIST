
# GAN and VAE with MNIST

Implementation of GAN and VAE on MNIST as well as Fashion MNIST


## Deployment

To run this project run

```bash
  git clone https://github.com/d3v-26/GAN-VAE-MNIST.git
  cd GAN-VAE-MNIST
  jupyter notebook
```

Following run all cells by: `Cell > Run All`
